# Bridging Genomics Project 

## Overview
The code and output accompanying "Inferring Sexual Network Bridging Using Gonorrhea Genomics: A Simulation Study" by Kline et al., in preparation. 

## Project Structure
- `code/` - associated parameters, scripts, and snakefiles 
- `output/` - simulation outputs
- `figures/` - manuscript figures
- `writeup/` - current draft of manuscript
